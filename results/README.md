# Results Directory

This directory contains outputs generated by the notebooks.

## Structure

```
results/
├── figures/     # PDF figures for paper
└── tables/      # CSV tables for data export
```

## Regenerating Results

Run the notebooks in order:
1. `notebooks/simulation.ipynb` - Monte Carlo simulations
2. `notebooks/empirical.ipynb` - LaLonde application

All figures are saved as PDF, all tables as CSV.
